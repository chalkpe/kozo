# KOZO

파이널판타지14 일본어 후리가나 오버레이

![Preview](public/preview.png)

## 기능

- 일본어 채팅에 포함된 한자에 후리가나를 표시
- 후리가나에 포함된 탁음 및 반탁음을 다른 색으로 표시
- 유저 채팅 및 시스템 메시지 필터링 및 다른 색으로 표시

## 설치

### 해루봇으로 연결

- **[해루봇](https://ffxivhaeru.tistory.com/notice)** → 기타 탭 → KOZO 연결하기

### 직접 추가

> https://kozo.chalk.moe/

1. `Advanced Combat Tracker` 실행
1. `Plugins` → `OverlayPlugin.dll` 탭으로 이동
1. `추가` 클릭 → `이름` 칸에 KOZO 입력
1. `프리셋` 값을 `커스텀`으로 설정 후 `OK` 클릭
1. 좌측에 추가된 KOZO 클릭 → `URL` 칸에 위 링크 복붙
